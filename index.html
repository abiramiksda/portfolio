Smart Resume Shortlisting System For College Placement

Project Overview

The Smart Resume Shortlisting System is an AI-powered web application developed to automate the resume screening and candidate shortlisting process for college placements. The system helps placement officers and recruiters efficiently analyze large numbers of resumes and identify the most suitable candidates based on job requirements.

Key Features

Automated resume upload and parsing

OCR support for scanned resumes

NLP-based skill extraction and job matching

Intelligent candidate ranking and shortlisting

Admin dashboard and analytics reporting

Secure candidate profile management

Technologies Used

Frontend: React.js

Backend: Python Flask

Database: MongoDB

Technologies: AI, NLP, OCR

Tools: VS Code, GitHub, Postman

How the System Works

Students upload resumes in PDF/DOC/DOCX formats.

OCR and NLP techniques extract skills, education, and experience.

Recruiters upload job descriptions.

The system compares resumes with job requirements.

Candidates are ranked based on matching scores.

The best candidates are automatically shortlisted.

Project Highlights

Reduced manual resume screening effort

Improved accuracy and fairness in recruitment

Supports large-scale placement drives

Fast and intelligent candidate filtering

Conclusion

This project demonstrates how Artificial Intelligence and Natural Language Processing can improve the efficiency of campus recruitment by automating resume analysis and candidate shortlisting.
